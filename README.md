# Titanic Dataset - Exploratory Data Analysis

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the Titanic Dataset using Python. The objective is to understand the structure of the dataset, identify missing values, classify numerical and categorical features, and gain meaningful insights before applying machine learning techniques.

---

## Objectives

- Load and inspect the dataset
- Understand dataset dimensions
- Analyze data types
- Identify missing values
- Separate numerical and categorical features
- Analyze unique values in categorical columns
- Detect duplicate records
- Generate statistical summaries
- Explore survival-related patterns

---

## Tools and Technologies Used

- Python
- Pandas
- NumPy
- Google Colab
- GitHub

---

## Dataset Information

The Titanic dataset contains information about passengers aboard the Titanic, including:

- Passenger ID
- Survival Status
- Passenger Class
- Name
- Gender
- Age
- Number of Siblings/Spouses
- Number of Parents/Children
- Ticket Number
- Fare
- Cabin
- Port of Embarkation

Dataset Size:

- Rows: 891
- Columns: 12

---

## Steps Performed

### 1. Dataset Loading

The dataset was loaded using Pandas.

### 2. Data Inspection

Displayed the first few records to understand the structure of the dataset.

### 3. Dataset Shape Analysis

Checked the number of rows and columns present.

### 4. Data Type Analysis

Examined the data type of each feature.

### 5. Statistical Summary

Generated descriptive statistics for numerical columns.

### 6. Missing Value Detection

Identified columns containing missing values.

### 7. Feature Classification

Separated features into:

- Numerical Features
- Categorical Features

### 8. Unique Value Analysis

Analyzed unique values present in categorical columns.

### 9. Duplicate Record Detection

Checked for duplicate rows within the dataset.

### 10. Survival Analysis

Observed survival distribution among passengers.

## Key Findings

### Missing Values

Missing values were found in:

- Age
- Cabin
- Embarked

### Numerical Features

Examples:

- Age

- Parch

### Categorical Features

Examples:

- Sex
- Cabin
- Embarked
- Ticket

### Survival Insights

- A significant number of passengers did not survive.
- Gender appears to have an impact on survival.
- Passenger class also influences survival probability.
